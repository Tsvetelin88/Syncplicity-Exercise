Syncplicity UI Automation Tests

**This project contains automated User Interface (UI) tests for the Syncplicity application, developed as an exercise to demonstrate best practices in C# and Selenium WebDriver.**

---

##  Overview
The project is built using .NET Core/Framework and leverages **Selenium WebDriver** for interacting with the web application and **NUnit** as the testing framework. The main goal is to automate core user flows, such as logging into the system (Login) and creating a new user (Create User).

## ⚙️ Installation and Setup

Follow the steps below to set up and run the tests locally:

## Clone the Repository
1.Open your command prompt (or Git Bash) and clone the repository to your local machine

2.Install Visual Studio(preferable) or another C# development environment

3.Fill the appsettings.json file with the provided credentials from the documentation

4.Open Test Explorer and Run LoginTests
